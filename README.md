# ISO_Calulator_Library

A collection of functions to calcualte pressure, temperature, density and speed of sound at different altitudes up to 85km, based on the International Standard Atmosphere

libary use:

isa isaCalc;   //Call first to initialise

//Returns floats   </br>
isaCalc.getAltTemp(altiude);  </br>
isaCalc.getAltPres(altiude);  </br>
isaCalc.getAltRho(altiude);  </br>
isaCalc,getAltSpeedOfSound(altiude);
